ABOVE, Autonomous Battle Above Endor, is an agent-based simulation of a battle between starships of the Rebel Alliance and those of the Galactic Empire from the well-known Star Wars movies.

It was originally written in 2004 as a 3D and AI playground by a group of Computer Science students at the University of Applied Sciences, Aargau, in Brugg-Windisch as project work for their 4th semester Software Engineering course.

The program is written in Java using Java3D, with the source code available from the GitHub project page.
